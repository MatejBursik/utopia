#+title: Utopia - Dungeon Adventure

Utopia is our game that we have written for Professional Skills 1.

* Installation

Game dependencies:

- ~pygame~
- ~pytmx~
- ~asyncio~

Install these with ~pip~.

* Usage
To run it:
#+begin_src sh
python main.py
#+end_src

OR

To run the game without downloading the code you can play it on our website:
- [[link][Game website]]
